# Operating system VH-DOS
### System Requirements
| System Component | At least this |
| :--------------- | ------------: |
| CPU | 186 |
| FPU | not required |
| Video Adapter | MDA/CGA/EGA |
| Hard disk capacity | 4 Kilobytes |
| PC Speaker | Preferably to have |
